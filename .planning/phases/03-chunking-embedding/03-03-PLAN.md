**Create pipeline/chunk_all.py** -- Orchestrator that reads all section files, chunks them, embeds them, and writes per-document JSONL output.

Key constants:
- `SECTIONS_DIR = Path("processed/sections")`
- `CHUNKS_DIR = Path("processed/chunks")`
- `BENCHMARK_RESULTS = Path("processed/benchmark/results.json")`

**Main function:**

`process_corpus(sections_dir: Path = SECTIONS_DIR, chunks_dir: Path = CHUNKS_DIR, skip_benchmark_check: bool = False) -> dict`:

Step 0: Pre-flight checks
- Verify Ollama is running and nomic-embed-text is available: call `get_model_version()`
- If not `skip_benchmark_check`: verify benchmark has passed by reading `BENCHMARK_RESULTS` and checking `passed == true`. If no benchmark results or benchmark failed, print error message and exit: "Benchmark has not passed. Run `python -m pipeline.benchmark` first."
- Get and store model version for metadata consistency (CHNK-07)
- Create `chunks_dir` if it doesn't exist

Step 1: Discover documents
- List all subdirectories in `sections_dir`
- Sort alphabetically for deterministic processing order
- Log: "Found {n} documents to process"

Step 2: Process each document
- For each document directory:
  1. Call `chunk_document(doc_dir)` to get list of ChunkRecords
  2. If no chunks produced, log warning and skip
  3. Call `embed_chunk_records(chunks, model="nomic-embed-text")` to add embeddings
  4. Write JSONL output file to `chunks_dir / f"{doc_id}.jsonl"`
  5. Log progress: "Processed {doc_id}: {n_sections} sections -> {n_chunks} chunks"

Step 3: Write JSONL
- For each document, write one JSONL file
- Each line: `json.dumps(chunk_record.model_dump())` with default=str for datetime/enum serialization
- Handle numpy float32 -> Python float conversion: use `.tolist()` on embedding vectors if they're numpy arrays

Step 4: Summary report
- Print to stdout:
  ```
  === Corpus Processing Complete ===
  Documents processed: {n}
  Total sections read: {n}
  Total chunks produced: {n}
  Total JSONL files written: {n}
  Embedding model: nomic-embed-text
  Embedding model version: {version}
  Output directory: processed/chunks/
  Errors: {n} (list details if any)
  ```
- Return summary dict with all stats

**Error handling:**
- If a single document fails (embedding error, file read error), log the error and continue with the next document. Do NOT stop the entire pipeline for one document failure.
- Track failed documents in a list and report at the end.
- If ALL documents fail, exit with error.

**CLI entry point:**
```python
if __name__ == "__main__":
    import sys
    import argparse

    parser = argparse.ArgumentParser(description="Chunk and embed the full corpus")
    parser.add_argument("--skip-benchmark-check", action="store_true",
                        help="Skip benchmark pass verification (for development only)")
    parser.add_argument("--sections-dir", type=Path, default=SECTIONS_DIR,
                        help="Input sections directory")
    parser.add_argument("--chunks-dir", type=Path, default=CHUNKS_DIR,
                        help="Output chunks directory")
    args = parser.parse_args()

    results = process_corpus(
        sections_dir=args.sections_dir,
        chunks_dir=args.chunks_dir,
        skip_benchmark_check=args.skip_benchmark_check,
    )

    if results.get("errors"):
        print(f"\nWARNING: {len(results['errors'])} document(s) had errors")
        sys.exit(1 if results.get("documents_processed", 0) == 0 else 0)
```

Important implementation notes:
- Import json, logging, argparse, Path at module level
- Import `chunk_document` from `pipeline.chunk`
- Import `embed_chunk_records`, `get_model_version` from `pipeline.embed`
- Use `logging.getLogger(__name__)` and configure with `logging.basicConfig(level=logging.INFO)` in __main__ block
- When serializing ChunkRecord to JSON: use `record.model_dump()` then `json.dumps(..., default=str)` to handle any non-serializable types
- For embedding vectors that may be numpy arrays: convert with `[float(x) for x in embedding]` or `embedding.tolist()` before JSON serialization
- Create `processed/chunks/.gitkeep` as part of directory creation

```python
import json
from pathlib import Path

chunks_dir = Path("processed/chunks")
jsonl_files = sorted(chunks_dir.glob("*.jsonl"))
print(f"JSONL files: {len(jsonl_files)}")

total_chunks = 0
model_versions = set()
errors = []

for jf in jsonl_files:
    doc_chunks = 0
    with open(jf) as f:
        for line_num, line in enumerate(f, 1):
            try:
                record = json.loads(line)
                assert record.get("text"), f"{jf.name}:{line_num}: empty text"
                assert len(record.get("embedding", [])) == 768, f"{jf.name}:{line_num}: bad embedding dim"
                meta = record.get("metadata", {})
                assert meta.get("embedding_model") == "nomic-embed-text", f"{jf.name}:{line_num}: wrong model"
                model_versions.add(meta.get("embedding_model_version", ""))
                assert meta.get("source_document"), f"{jf.name}:{line_num}: missing source_document"
                doc_chunks += 1
            except Exception as e:
                errors.append(str(e))
    total_chunks += doc_chunks
    print(f"  {jf.name}: {doc_chunks} chunks")

print(f"\nTotal chunks: {total_chunks}")
print(f"Model versions: {model_versions}")
print(f"Errors: {len(errors)}")
assert len(model_versions) == 1, f"Multiple model versions found: {model_versions}"
assert len(errors) == 0, f"Errors found: {errors[:5]}"
print("\nINTEGRITY CHECK PASSED")
```
