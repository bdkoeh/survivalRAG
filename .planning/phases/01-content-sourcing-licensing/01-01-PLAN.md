**download-all.sh** must:
1. Use `set -euo pipefail` for safety
2. Be idempotent -- skip files that already exist (check file presence)
3. Use curl with `--retry 3 --retry-delay 5 -fSL` flags for reliability
4. Include ALL document URLs from the research Download URL Registry (see 01-RESEARCH.md)
5. Handle fallback URLs -- if primary fails, try fallback
6. Print clear status for each file (SKIP/DOWNLOADING/OK/ERROR)
7. Generate SHA-256 checksums after all downloads complete, writing to sources/checksums.sha256
8. Use sha256sum on Linux/Mac; on Windows Git Bash, sha256sum is available

3. For any failed downloads:
   - Check if the primary URL returned a non-200 status (site restructured, funding lapse, etc.)
   - Try the fallback URL if available

   - Military FMs should each be 100+ KB minimum
